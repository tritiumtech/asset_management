# 资产管理系统

**项目代号**: AssetMgmt  
**目标**: 基础架构部资产数字化管理，替代传统Excel管理，实现资产全生命周期管理

---

## 项目概述

基于飞书多维表格 + AI 助手的轻量级资产管理系统，无需开发资源即可快速上线。

## 目录结构

```
asset_management/
├── README.md           # 项目说明
├── doc/                # 文档目录
│   └── PRD.md         # 产品需求文档
└── src/               # 源码目录（如有）
```

## 快速开始

### Phase 1: 飞书多维表格方案（推荐）
- 使用飞书多维表格作为数据存储
- 通过飞书审批流实现领用/归还流程
- AI 助手提供查询和统计功能

### Phase 2: 增强功能（可选）
- 开发独立管理后台
- 集成 RFID/条码扫描
- 自动化盘点
